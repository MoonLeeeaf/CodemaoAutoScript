### 使用

安装 Node.js

然后下载你所需要的脚本, 放在任意目录

新建文件 cookie.txt 登录 shequ.codemao.cn

通过任意方法获取作用域为 .codemao.cn 的 Cookie 并粘贴进去（截至今天，只需要 authxxxxx 即可）

之后运行脚本即可
